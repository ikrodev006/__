# ikro.dev

Personal portfolio — minimal, fast, and native-feeling.

## Stack

- **React** + **Vite**
- **Tailwind CSS v4**
- **Framer Motion** — spring-based animations
- **New York** serif typography (Apple ecosystem)

## Features

- Animated splash screen with continuous motion transition
- Typewriter effect on subtitle
- Phrase carousel (slide on mobile / fade on desktop)
- Glassmorphism design system
- Dark mode by default
- Fully responsive

## Run locally

```bash
npm install
npm run dev
```
